02

Pendulum Learning Model for Visually Impaired Students

This project is an assistive learning model designed to help visually impaired students understand pendulum motion. It uses a curved tactile path to represent the swing of a pendulum, allowing students to feel and imagine its movement. A sensor detects oscillations and a buzzer provides sound feedback, helping students understand the concept of motion and time period through touch and audio. The model promotes inclusive learning by making physics concepts more accessible without relying on visual observation.

03

Hackathon ARNav

AR-Based Navigation System for the Visually Impaired Developed an assistive mobile application that helps visually impaired people navigate their surroundings safely using Augmented Reality (AR) and Artificial Intelligence. The app uses the YOLOv8 object detection model to detect obstacles such as stairs, walls, and other objects in real time through the device camera. It provides guidance through an integrated voice assistant that communicates navigation instructions and alerts to the user. The application focuses on accessibility, offline functionality, and user safety while reducing battery consumption. This project demonstrates the use of computer vision, AR technology, and voice interaction to create an inclusive and accessible navigation solution.
